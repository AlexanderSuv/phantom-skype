# phantom-skype
Phantom wrapper for [Skype](https://www.skype.com)

## Compatibility

* Node.js `6.2.1`

## Install

```js
npm install -g
```

## Usage

```
phantom-skype
```
```
[processor] initialized
phantomSkype>
```
## Command list
* login
* recipient
* message
* status
* pageStatus (creates a screenshot of current page at '/screenshots/status.png')
